# Contributing to Resona

First off, thank you for considering contributing! It’s people like you who make open source such a great community.

## 🚀 How Can I Contribute?

### Reporting Bugs

If you find a bug, please open an **Issue** and include:

- A clear, descriptive title.
- Steps to reproduce the problem.
- What you expected to happen vs. what actually happened.
- Your environment (OS, Browser, etc.).

### Suggesting Enhancements

We are always open to new ideas! Please open an issue first to discuss the change before you start coding, so we can ensure it aligns with the project goals.

---

## 🛠️ Development Workflow

To keep the project history clean and organized, please follow these steps:

1. **Fork** the repository and create your branch from `main`.
2. **Name your branch** according to the type of work you are doing (see below).
3. Make your changes and ensure the code follows the existing style.
4. **Push** to your fork and submit a **Pull Request**.

### Branch Naming Conventions

Please prefix your branch names based on the following categories:

| Category          | Prefix            | Example                     |
| :---------------- | :---------------- | :-------------------------- |
| **New Features**  | `feat/`           | `feat/user-login-system`    |
| **Bug Fixes**     | `fix/`            | `fix/header-alignment`      |
| **Refactoring**   | `refactor/`       | `refactor/api-logic`        |
| **Improvements**  | `perf/` or `enh/` | `perf/load-speed`           |
| **Maintenance**   | `chore/`          | `chore/update-dependencies` |
| **Documentation** | `docs/`           | `docs/update-readme`        |

---

## 📝 Pull Request Guidelines

- Keep your PRs focused. If you have two unrelated fixes, please submit two separate PRs.
- Mention the Issue number if your PR fixes a specific bug (e.g., "Closes #12").
- Be patient! We will review your code as soon as possible.

## ⚖️ Code of Conduct

By participating in this project, you agree to maintain a respectful and welcoming environment for everyone.
